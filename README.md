To Do List Web Application adalah aplikasi berbasis web yang digunakan untuk mencatat, mengelola, dan memantau kegiatan harian pengguna. Aplikasi ini dirancang dengan tampilan sederhana namun interaktif, sehingga memudahkan pengguna dalam menambahkan, melihat, mencari, memfilter, dan menghapus kegiatan berdasarkan tanggal tertentu.

Project ini dibuat sebagai latihan pengembangan web front-end dengan fokus pada manipulasi DOM, penyimpanan data lokal (Local Storage), serta penggunaan framework CSS modern untuk mempercantik tampilan.
